🔧 REX開発ツール (PWA対応)

※ PWA対応: スマホ・タブレットからも使用可能です
使い方:
1. 各項目を入力（バージョン番号は自動で+1されます）
2. HEAD_COMMIT（今回のフルSHA）とBASE_COMMIT（前回のフルSHA）を必ず入力
3. BUILD_IDを入力すると、検証URLに自動で追加されます
4. 「報告書を生成」ボタンをクリック
5. 生成されたMarkdownをコピーまたはダウンロード

重要: compare URLは固定SHAで生成されます（HEAD~1...HEAD形式は禁止）
必須: HEAD_COMMITとBASE_COMMITの両方を必ず入力してください（フルSHA推奨）
注意: このHTMLファイルをダブルクリックしてもVisual Studioが開く場合は、
ファイルを右クリック → 「プログラムから開く」 → ブラウザを選択してください。
